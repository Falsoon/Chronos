A prototype to add interactive character-graphics to augment the INFORM source code, and to simplify the INFORM source code using forms-based input.

Test Project was created with Inform and has IFID 6F28FB2E-0793-42A7-B85C-451D83682A1B. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
